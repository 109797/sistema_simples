Sistema Simples de Cadastro e Listagem de Produtos
Este é um sistema básico de CRUD (Create e Read) para cadastro e listagem de produtos, desenvolvido em PHP puro com MySQL, sem frameworks ou orientação a objetos. Ideal para estudos de faculdade ou projetos simples.

Tecnologias Utilizadas
PHP (procedural com mysqli)
MySQL (banco de dados)
HTML/CSS (interface responsiva e moderna)
Como Acessar o Sistema
Instale o XAMPP (ou similar) e inicie Apache e MySQL.
Clone ou baixe este repositório e coloque a pasta sistema_simples em htdocs.
Crie o banco de dados: importe o arquivo database.sql no phpMyAdmin ou execute o SQL no terminal MySQL.
Acesse no navegador: http://localhost/sistema_simples/index.php.
O sistema cria automaticamente o banco e tabela se não existirem. Navegue entre as páginas para cadastrar e listar produtos.
